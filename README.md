# RouteShare
Marchenko_OOP_Project

Тема організації спільних поїздок є дуже актуальною у сучасному світі. Через зростання цін на пальне люди шукають способи заощадити на транспорті. Сервіси карпулінгу вирішують цю проблему: вони допомагають водіям покрити частину витрат на дорогу, а пасажирам — подорожувати дешевше та комфортніше, ніж автобусами чи потягами.
Потенційними користувачами такого програмного забезпечення є дві основні групи людей: водії, які планують поїздку і мають вільні місця в автомобілі, та пасажири, яким потрібно швидко дістатися до певного пункту призначення.
Серед програмних аналогів у цій сфері лідерами є:
•	BlaBlaCar. 
•	FlixBus.
•	Uber (функція спільних поїздок).
Для розробки даного програмного забезпечення буде використано такий стек технологій:
•	Мова програмування: C# з використанням об'єктно-орієнтованої парадигми, LINQ та делегатів. 
•	Платформа: .NET для створення десктопного застосунку.
•	Збереження даних: серіалізація та десеріалізація об'єктів у локальні файли у форматі JSON.
